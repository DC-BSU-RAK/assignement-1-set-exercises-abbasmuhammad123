HTML & CSS Quick Reference

Essential guide for HTML & CSS

Introduction

Quick overview of HTML tags and CSS properties with examples.

HTML Tags

<html>

Root element

<html lang="en">...</html>

<head>

Meta info & links

<head>
    <meta charset="UTF-8">
    <title>Page</title>
</head>

<body>

Visible content

<body>
    <h1>Welcome</h1>
    <p>Content</p>
</body>

Headings

Levels h1-h6

<h1>Main</h1>
<h2>Sub</h2>

<p>

Paragraph

<p>Text here</p>

<a>

Hyperlink

<a href="url">Link</a>

<img>

Image

<img src="img.jpg" alt="desc">

Lists

Ordered & unordered

<ul>
    <li>Item</li>
</ul>

<div> & <span>

Containers

<div>Block</div>
<span>Inline</span>

Table

Data tables

<table>
    <tr>
        <th>Head</th>
        <td>Data</td>
    </tr>
</table>

Forms

User input

<form>
    <input type="text">
    <button>Send</button>
</form>

<pre> & <code>

Code snippets

<pre><code>Code</code></pre>

Semantic

Meaningful tags

<header>...</header>
<main>...</main>

CSS Properties

Color

color: #333;
background: #f5f5f5;

Text

font-family: Arial;
font-size: 16px;
text-align: center;

Box

width: 100%;
padding: 20px;
margin: 10px;
border: 1px solid;

Layout

display: flex;
position: relative;

Flexbox

.container {
    display: flex;
    justify-content: center;
}

Grid

.container {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

Animation

transition: all 0.3s;
animation: slide 1s;

Media

@media (min-width: 768px) {
    body { font-size: 18px; }
}
